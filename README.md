Project Allocation System (Full-Stack Web Application)
Spring Boot, React.js, JWT, OAuth 2.0, MySQL
• Designed and developed a scalable web application to automate student project allocation with fairness and
transparency
• Built secure REST APIs using Spring Boot integrated with Spring Security
• Implemented JWT-based authentication and OAuth 2.0 (Google Sign-In) for secure login and session man-
agement
• Developed dynamic frontend using React.js and Axios for seamless API communication
• Designed allocation algorithm to assign students based on preferences while preventing over-allocation
• Implemented automatic team formation and project grouping functionality
• Integrated email service for notifications including allocation confirmation and updates
• Built an interactive chatbot to assist users and improve system usability
• Deployed frontend on Vercel and backend on Railway/Render for production usage
